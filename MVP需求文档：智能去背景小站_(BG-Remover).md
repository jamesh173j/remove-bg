# MVP 需求文档：智能去背景小站 (BG-Remover)

## 1\. 产品定义与核心价值

* **产品定位：** 一个极简的、基于 AI 的在线图片去背景工具。  
* **核心假设：** 用户需要一个**无需注册、即用即走、隐私安全**（不存储图片）的高质量抠图工具。  
* **目标用户：** 电商卖家、需要制作证件照的学生、PPT 重度使用者。

## 2\. 核心功能范围 (Scope)

### 【必须有 \- Must Have】

* **图片上传：** 支持拖拽或点击上传（格式：JPG/PNG，大小 \< 10MB）。  
* **自动抠图：** 调用 Remove.bg API 进行背景擦除。  
* **结果预览：** 在线对比原图与去背景后的效果。  
* **无痕下载：** 下载透明背景的 PNG 图片到本地。  
* **内存暂存：** 图片仅在处理时驻留内存，下载后或关闭页面立即释放。

### 【暂不做 \- Won't Have】

* 用户注册与登录系统。  
* 历史记录保存。  
* 在线手动修补（Eraser/Restore）。  
* 批量处理。

---

## 3\. 用户路径 (User Flow)

1. **访问首页：** 看到上传区域。  
2. **上传图片：** 后端 API 自动调用，前端展示进度。  
3. **获取结果：** 预览效果，点击“下载”按钮，任务结束。

---

## 4\. 业务逻辑与规则

* **存储规则：** 内存中转模式，后端不进行任何硬盘存储。  
* **安全限制：** 增加前端限流，防止恶意消耗 API 余额。  
* **隐私声明：** 显著标注不存储任何用户图片。

---

## 5\. 验证指标 (Success Metrics)

* **任务完成率：** 下载次数 / 上传次数。  
* **响应时间：** 平均处理时长。
